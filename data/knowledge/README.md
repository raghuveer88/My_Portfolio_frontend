# Knowledge files for the chatbot

Update these whenever your Resume or LinkedIn changes.

## Files
- `profile.json`  : curated highlights + speaking style
- `resume.txt`    : paste extracted resume text (from PDF)
- `linkedin.txt`  : paste extracted linkedin export text (from PDF)

## Tip (recommended)
Keep PDFs in your repo for humans if you want, but the chatbot reads only the `.txt` files.
